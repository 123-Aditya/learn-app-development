**Learn-App-Development**

Don’t overthink stack.

👉 Use this:
	•	Java
	•	Android Studio
	•	Simple UI
	•	One idea → publish fast

Your goal should be:

“Ship 1 app on Play Store ASAP”

---

**Steps:**
- Install Android Studio
- Learn
    - Activity
    - Layout (XML)
    - Buttons, TextView
- Navigation + Multiple Screens
- API calls
- Build Real App + publish

---

**How to Publish on Play Store**

Steps:
- Create Google Play Developer account - One-time fee: $25 (~₹2000)

- Build signed APK / AAB
  
- Upload on Google Play Console
  
- Add:
		- App name
		- Screenshots
		- Description
		- Icon
  
- Submit for review

⏱ Approval time: ~1–3 days
